A Common Layout in Next.js is used to maintain consistent UI elements like header, footer, and sidebar across multiple pages.

It avoids code duplication and improves maintainability.

In the App Router, layouts are created using the layout.js or layout.tsx file.

The layout file wraps all child pages using the {children} prop.

Common layouts are useful for navigation bars, authentication wrappers, and dashboards.

Nested layouts allow different sections of the app to have their own structure.

Layouts persist between page navigations, improving performance.

Styling can be applied globally using CSS or Tailwindcss and Bootstrap also.

Helps in building scalable and clean Next.js applications. Next JS is React Framework
